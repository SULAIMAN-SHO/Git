أولاً: تهيئة المشروع كمستودع جديد
git init

ثانياً: عرض حالة الملفات (المتابعة وغير المتابعة)
git status

ثالثا: إضافة ملف محدد (مثلاً index.html) للمتابعة
git add index.html

رابعاً: إيقاف متابعة ملف معيّن بدون حذفه من الجهاز
git rm --cached style.css

خامساً: إضافة جميع الملفات الموجودة في المشروع
git add .

سادساً: حفظ التغييرات مع رسالة توضيحية
git commit -m "أول حفظ للتغييرات"

إعداد اسم المستخدم العالمي في Git
git config --global user.name "SULAIMAN"

إعداد البريد الإلكتروني العالمي في Git
git config --global user.email "sulaiman.lutfi.sho@gmail.com"

عرض اسم المستخدم الحالي في إعدادات Git
git config user.name

عرض البريد الإلكتروني الحالي في إعدادات Git
git config user.email
